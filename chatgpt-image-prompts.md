## Color Reference

For all graphics:
- Primary Red: #DC143C (crimson)
- Dark Red: #8B0000 (darker crimson for gradients)
- Gold/Yellow: #FFD700 (bright gold)
- White: #FFFFFF
- Black: #000000
- Dark Gray 1: #1a1a1a
- Dark Gray 2: #2d2d2d

## Typography

- Headlines: Bold, sans-serif (Arial/Helvetica style)
- Body text: Regular weight sans-serif
- Quote marks: Serif font (Georgia style)
- All text should be clear, high contrast, and readable at small sizes

## Export Settings

Request PNG format with:
- High resolution (300 DPI if possible)
- Transparent background NOT needed (solid backgrounds specified)
- RGB color mode
- Web-optimized file size

1. Copy each prompt individually into ChatGPT (with DALL-E access)
2. Request PNG format output
3. If colors or layout don't match exactly, provide specific correction prompts
