### Converting SVG Files
SVG files are vector graphics that scale perfectly. To use them:

**For Social Media:**
1. Open in any modern browser
2. Take a screenshot, OR
3. Use an online converter (CloudConvert, Convertio) to make PNG/JPG
4. Recommended: Export at 2x size for high-quality

**For Editing:**
- Import into Figma (free online)
- Edit in Adobe Illustrator
- Use Inkscape (free open-source)
- Edit in Canva (supports SVG import)

**For Printing:**
- SVGs scale infinitely - perfect for posters, banners
- Export to high-res PDF for print shops
